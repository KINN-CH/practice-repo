# Contributing to Mobility-Insight AI

저희 프로젝트에 관심을 가져주셔서 감사합니다! 기여를 원하신다면 아래 절차를 따라주세요.

1. **이슈 생성**: 버그나 제안 사항은 Issue 탭에 먼저 등록해 주세요.
2. **브랜치 규칙**: `feature/기능이름` 형식을 사용합니다.
3. **Pull Request**: `main` 브랜치로 PR을 보낸 후, 1명 이상의 리뷰어 승인을 받아야 합니다.
4. **커밋 메시지**: Conventional Commits 규격을 준수해 주세요. (예: `feat: add new sensor module`)
